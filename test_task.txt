Тестовое задание


Требования к технологиям:

•	Python 3.10 +
•	Django, Django Ninja
•	Postgresql


Описание:

Разработать бэкенд для блога. Необходимо реализовать следующий функционал:

•	Регистрация: пользователь вводит username и password. На сервере генерируется токен - строка из 256 рандомных символов.
•	Авторизация: при каждом запросе передаётся токен в заголовках или body. Бек должен проверять, существует ли пользователь с таким токеном в базе, и если да, то предоставлять доступ (будет плюсом, если подключишь библиотеку Django ninja jwt).
•	Добавить ендпоинты на просмотр, cоздание, обновление, удалений статей. Предусмотреть, что обновлять и удалять пользователь может только свою статью.
•	Добавить ендпоинты на просмотр, создание, обновление, удаление комментариев к статье. Предусмотреть, что обновлять и удалять пользователь может только cвой комментарий.
•	Есть возможность управления пользователями, статьями, комментариями и категориями через админку. 

Требования к приложению:

•	Подключен postgresql
•	Все данные необходимо отдавать в json формате, views писать не нужно
•	На post запросах должны присутствовать валидации, как минимум на required
